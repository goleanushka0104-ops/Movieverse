# Movieverse
# 🎬 MovieVerse – Smart Movie Recommendation Website

MovieVerse is a modern movie recommendation web application built using HTML, CSS, and JavaScript.

The platform helps users discover movies based on genres, moods, and search queries while providing a clean cinematic UI experience with trailer support and favorites management.

Designed with an interactive glassmorphism interface, MovieVerse combines entertainment and modern frontend development concepts into a single responsive website.

---

# 🚀 Features

### 🔍 Smart Movie Search

Search movies instantly using the OMDb API integration.

### 🎭 Mood-Based Recommendations

Get movie suggestions based on emotions like:

* Happy
* Sad
* Love
* Scary
* Space / Sci-Fi

### ❤️ Favorites System

Save favorite movies using Local Storage.

### 🎥 Trailer Support

Watch movie trailers directly through YouTube integration.

### 📱 Responsive Design

Works on desktop and mobile devices.

### 🎨 Modern UI/UX

Glassmorphism design with cinematic dark theme and smooth layout.

### 📂 Multiple Genres

Explore movies from:

* Action
* Drama
* Sci-Fi
* Romance
* Horror

# 💡 How It Works

1. Users can browse movie categories.
2. Search functionality fetches movies dynamically.
3. Mood recommendation system suggests movies based on user emotions.
4. Users can save movies to Favorites.
5. Clicking movie posters opens trailers on YouTube.
